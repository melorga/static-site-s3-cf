# Security Policy

## Supported versions

This repository tracks `main`. Security fixes land on `main` and are not back-ported to historical commits.

## Reporting a vulnerability

Please report suspected vulnerabilities privately via GitHub Security Advisories:

1. Open https://github.com/melorga/static-site-s3-cf/security/advisories/new
2. Provide reproduction steps, the impacted Terraform/AWS resources, and any suggested mitigation.

We aim to acknowledge new reports within 5 business days. Please do not open public issues for security problems.
